# Retroserver

Lightweight Ruby Web Server

## Installation

Add this line to your application's Gemfile:

    gem 'retroserver'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install retroserver

## Usage

Add this line to your application's primary .rb file

    require 'retroserver'
